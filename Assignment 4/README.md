# 🎨 AI Image Studio

**Assignment 4 – Upgrading the AI Image Studio**  
**MirAI School of Technology | Virtual Summer Internship 2026 | AI Builder Track**

AI Image Studio is a modern, single-page Streamlit application designed for generating high-quality AI artwork through the Pollinations AI Image API. The application focuses on a clean user experience, responsive controls, and enhanced image generation features within a professional SaaS-inspired interface.

---

## ✨ Highlights

- 🖼️ AI-powered image generation using **Pollinations AI**
- 🎨 10 curated artistic styles
- 📐 Dynamic image resolution controls
- ✨ Magic Enhance mode for higher-quality prompts
- 🎲 "Surprise Me" random prompt generator
- 📥 Smart PNG download with dynamic filenames
- ⚡ Fast and lightweight single-page interface
- 📱 Responsive, modern UI with smooth user experience

---

## 🚀 Features

- Generate AI artwork from custom text prompts
- Real-time width and height customization
- Automatic prompt enhancement for improved image quality
- Multiple professional art styles
- One-click random creative inspiration
- Proper image download with `.png` extension
- User-friendly layout optimized for desktop viewing
- Robust error handling for failed image requests

---

## 🛠️ Built With

- **Python 3.10+**
- **Streamlit**
